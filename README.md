# pyyoga
A python wrapper for the Yoga layout engine by Facebook (https://facebook.github.io/yoga/).

## Dependencies

Cython (tested with 0.29.19)

## Installation

Fetch source code of Yoga:

```
$ git submodule init
$ git submodule update
```

Run build:

```
$ make install
```

View on `tests.py` for details on API call
